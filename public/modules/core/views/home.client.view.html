wARriorDrone Project

Jeff Brookshaw @Winona State University

About

WSU Undergraduate Research Seminar Poster Presentation

Welcome the wARriorDrone project! The wARriorDrone web application enables any AR. Drone 2.0 to track and follow colored objects with a user friendly interface, that can be anything from a t-shirt to a frisbee as shown in the videos. The application is built using node.js to send flight commands and stream data from the AR. Drone's sensors, Angular.js is used to bind control variables in the front end to the back end drone commands.

Presentations

The application was created for the Winona State University Undergraduate Research Seminar, I presented my application on two separate occasions. First at the WSU under graduate research poster presenation conference, I then preformed a live demo for a full house at the WSU Computer Science Research Conference for which I received the CS award for best research.
